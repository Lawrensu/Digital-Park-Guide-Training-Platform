# Digital Park Guide Training Platform

This project is developed for Sarawak Forestry Corporation to provide a digital training platform for park guides.

The system includes:
- Web application (React + Vite)
- Mobile application (React Native + Expo)
- Backend services for data, authentication, and AI integration

## Features
- Interactive training modules
- Progress tracking and certification
- Secure authentication and data protection
- AI-based abnormal activity detection using IoT devices

## Project Structure
- /backend → API and server logic
- /frontend/web → Web application
- /frontend/mobile → Mobile application
- /docs → Documentation and planning

## Objective
To provide a flexible, secure, and scalable platform for training and monitoring park guides.
