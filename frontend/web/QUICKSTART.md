# ⚡ Quick Start Guide

Get the Digital Park Guide Training Platform web app running in 2 minutes!

## 1️⃣ Install Dependencies

```bash
cd frontend/web
npm install
```

## 2️⃣ Start Development Server

```bash
npm run dev
```

Browser opens automatically at: **http://localhost:5173**

## 3️⃣ Login

- **Email**: `guide@example.com`
- **Password**: `password123`

Or enter any email and password to create a new session.

## ✅ Done!

You can now:

- 📊 **Dashboard** - See learning progress
- 📚 **Courses** - Browse & search courses
- 🎬 **Lessons** - Watch videos
- 📝 **Quizzes** - Test your knowledge
- 🏆 **Certificates** - View achievements

## 🎨 Quick Customizations

### Change Colors
Edit `tailwind.config.js` line 7-11

### Add a Course
Edit `src/data/courses.js`, add to courses array

### Change App Name
Edit `index.html` title tag

## 🚀 Build for Production

```bash
npm run build
npm run preview
```
